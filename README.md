## Installation
:warning: Requirements: Freeglut

1. Clone this repository somewhere on your machine
2. Go to that directory
3. Type `make`
4. Run with `./3pt`

## :video_game: Controls 

| **Key**  | ** Action** |
|------|--------|
|s    | Shoot  |
|1 - 5| Change position |
|  + |  Angle ++ |
|  - |  Angle - - |
|  * | Start velocity ++ |
|  / | Start velocity - -|
| Esc| Exit |
